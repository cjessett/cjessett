#### 🥬 [FarmSignal](http://farmsignal.vercel.app)
App connecting farmers and chefs — farmers update produce availability and an iOS Shortcut sends automated messages directly
from their phone. Includes tiered features and a subscription model. Built with Next.js, PostgreSQL, OAuth, and Stripe.

#### 🌤️ [Lapse Rate](https://cjessett.com/lapse-rate)
A weather forecasting tool for pilots, paragliders, hang gliders, and atmospheric enthusiasts that computes the environmental lapse rate using real-time forecast data from Open-Meteo. 


#### 🎧 [Narrate](https://github.com/cjessett/narrate)
Convert PDF and EPUB files to natural-sounding audio using AI TTS. Supports self-hosted Kokoro, OpenAI TTS, ElevenLabs, and Google Cloud TTS — switch providers by changing a single env var.
